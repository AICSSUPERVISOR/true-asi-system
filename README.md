## 3. Getting Started

### 3.1. Prerequisites

- Python 3.11+
- AWS Account with credentials
- GitHub account

### 3.2. Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/AICSSUPERVISOR/true-asi-system.git
   cd true-asi-system
   ```

2. **Install dependencies**:
   ```bash
   pip3 install -r requirements.txt
   ```

3. **Configure environment**:
   - Copy `.env.example` to `.env`
   - Fill in your AWS, OpenAI, Firecrawl, and Manus API keys

### 3.3. Running Demonstrations

- **Self-Improvement Demo**:
  ```bash
  python3 run_self_improvement.py
  ```

- **Distributed Framework Demo**:
  ```bash
  python3 run_distributed_demo.py
  ```

- **Reasoning Engines Demo**:
  ```bash
  python3 run_reasoning_demo.py
  ```

- **Massive Processing Demo**:
  ```bash
  python3 run_massive_processing.py
  ```

### 3.4. Accessing S3 Data

Full instructions for accessing the 19.02 GB of data in S3 are available in `AWS_S3_DATA_README.md`.
