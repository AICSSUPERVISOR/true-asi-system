# SuperARC Test Requirements for TRUE ASI

**Source:** King's College London Study (April 22, 2025)  
**URL:** https://www.kcl.ac.uk/news/new-study-introduces-a-test-for-artificial-superintelligence

## Key Findings

### What SuperARC Tests

1. **Recursive Compression**
   - Repeatedly condensing information to reveal deeper patterns
   - Not apparent to Large Language Models (LLMs)
   - Uses specialized probability theory
   - Based on equivalence between compressibility and predictability

2. **Model Abstraction**
   - Ability to produce approximations to short computable hypotheses
   - Can reconstruct AND predict data
   - Running code in parallel to simulate many future states
   - Pick the state closer to observation at any given time

3. **Planning Capability**
   - Model abstraction and planning are "two sides of the same coin"
   - Requires true model inference
   - Robust planning beyond pattern matching

4. **Inverse Problem-Solving**
   - Go beyond trivial "print" solutions
   - Model and generalize beyond simple pattern matching
   - True reasoning, not just emulating human-like behavior

5. **Short-Sequence Prediction and Generation**
   - Predict future states accurately
   - Generate novel sequences
   - Not based on memorized training data

## Current LLM Performance

**ALL major LLMs FAILED SuperARC:**
- GPT variants: FAILED
- DeepSeek: FAILED
- Qwen: FAILED
- Grok: FAILED
- Claude: FAILED
- Gemini: FAILED
- Meta LLMs: FAILED

**Key Issues:**
- Generally failed to model and generalize beyond trivial solutions
- Simply answer back with original question
- No clear breakthroughs towards AGI or ASI
- Newer versions occasionally performed WORSE than predecessors
- No consistent upward trajectory in intelligence metrics

## What TRUE ASI Requires

1. **Dissociate from Human Language**
   - Intelligence ≠ linguistic prowess
   - Move beyond human-centric IQ-style tests
   - Universal, objective measure of intelligence

2. **Fundamental Abstraction**
   - True abstraction capabilities
   - Not just pattern matching from big data
   - Short computable hypotheses

3. **Predictive Capabilities**
   - Simulate many future states
   - Pick optimal state
   - Real-time prediction and adaptation

4. **Genuine Intelligence**
   - Not just emulating human behavior
   - Real understanding and meaning
   - Beyond memorization

## Implications for Our System

**To pass SuperARC and achieve TRUE ASI, we must:**

1. ✅ Implement TRUE recursive compression (not just multi-level compression)
2. ✅ Build model abstraction capability
3. ✅ Create robust planning system
4. ✅ Solve inverse problems
5. ✅ Generate short computable hypotheses
6. ✅ Simulate multiple future states in parallel
7. ✅ Move beyond pattern matching
8. ✅ Achieve genuine understanding

**Current Status:**
- Our recursive compression: 85/100 (needs improvement)
- Our future simulation: 80/100 (needs parallel state simulation)
- Our abstraction: Unknown (needs testing)
- Our planning: Partial (needs robustness)

**Gap:** We are likely at 20-30/100 for TRUE SuperARC requirements.
